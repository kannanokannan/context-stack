# ContextOps

ContextOps answers: how does an organisation govern its AI context?

It owns:

- context governance framework
- People, Process, Context triad
- Capture, Curate, Supply, Renew spine
- maturity model
- named practices and governance roles

It does not own egress tier policy, runtime implementation, website navigation, or MCP server code.
