# Egress Tier

Egress Tier classifies where data is permitted to flow.

Locked tier numbering:

- Tier I: never leaves the system; most restrictive.
- Tier II: anonymised aggregates only, with consent.
- Tier III: explicit per-call API escalation; least restrictive.

Use `Egress Tier`. Do not use `Privacy Tier`.
